Requirements

  • Tener un computadora con conexión a Internet para descargar las herramientas requeridas.
  • Tener ganas de APRENDER.

Description

Bienvenidos/as al curso mas completo que podrás encontrar sobre SQL. En este curso aprenderás desde lo mas básico como crear Bases de Datos, Tablas e Insertar Registros hasta temas mas complejos como Procedimientos Almacenados y Triggers.

En este curso utilizaremos el gestor de Base de Datos MYSQL es el #1 utilizada por los desarrolladores y las empresa. Los conocimientos obtenidos de este curso te podran servir en otros gestores de base de datos como son ORACLE, SQL-SEVER, POSTGRESQL, FIREBIRD y muchos mas.

Todo en este curso cuenta con una explicación y ejercicio practico en el mismo vídeo.

El curso contara con 14 secciones las cuales son las siguientes:

  1. Bienvenida.

  2. Mostrar-Crear-Eliminar Bases de Datos

  3. Mostrar-Crear-Eliminar Tablas

  4. Cargando Registros

  5. Agregando valor a las Tablas

  6. Tipos de Datos

  7. Funciones

  8. Indices

  9. Alter Table

  10. Join

  11. Subconsultas

  12. Vistas

  13. Procedimientos Almacenados

  14. Disparadores (TRIGGERS)

¿Quien debería tomar este curso?

  • Cualquier persona interesada en aprender sobre SQL para el manejo de bases de datos.

  • A todo aquel que requiera incorporar una Base de Datos a una Aplicación, Sitio Web.

NOTA:

Si ya eres una persona que administras Base de Datos este curso NO ES PARA TI, los temas que tocaremos aquí serán básicos-avanzados pero no sera tan complejo.

Who this course is for:

  • Principiantes en el manejo de SQL.
  • Personas que quieran aprender temas básicos y avanzados de SQL.
